Manage Blogs

Sign in with a GitHub token to create and publish new blog posts directly to eastvillagebuyers.com.

Needs a fine-grained token scoped only to the evb-site repo, with Contents: Read and write permission. Create one at github.com/settings/personal-access-tokens. The token is stored only in this browser (localStorage) and is sent directly to GitHub's API — never to any third-party server.

New Blog Post

Fill this in and publish — it builds the post using the site's existing template, adds it to the blog index, and updates the sitemap.

Image must already be uploaded to the root of the site repo (same place as your other product photos).

Existing Posts

Pulled live from the blog/ folder in your repo.
